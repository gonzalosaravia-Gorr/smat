# SMAT_PROYECTO

## 📌 Descripción

Aplicación desarrollada con FastAPI (backend) y Flutter (frontend) que permite visualizar estaciones registradas.

## 🚀 Tecnologías

* Python (FastAPI)
* Flutter (Dart)
* SQLite

## ▶️ Ejecución

### Backend

```bash
cd backend
uvicorn app.main:app --reload
```

### Frontend

```bash
cd mobile
flutter run -d chrome
```

## 🌐 Endpoints

* `/estaciones/` → lista de estaciones
